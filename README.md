# Train Station Signage Accessibility Analysis

## Overview

This project conducted a multi-city spatial analysis of train station directional road signages and their relationship to pedestrian accessibility, aiming to identify optimal signage placement strategies for improving urban wayfinding and accessibility. The study compares train station signage systems across three international cities:

- Hong Kong
- London
- Auckland

The project applies GIS spatial analysis and network analysis techniques using ArcGIS Pro to evaluate how directional signages assist pedestrians in accessing train station entrances efficiently.

---

## Research Objectives

The main objectives of this project are:

- Analyse the spatial distribution patterns of train station directional signages
- Evaluate the effectiveness of signage placement for pedestrian accessibility
- Compare signage systems between different cities and station environments
- Identify spatial characteristics associated with effective wayfinding systems
- Provide recommendations for optimized train station directional signage design

## Methodology

The project uses both spatial analysis and network analysis approaches.

### Spatial Analysis
- Signage Quantity Analysis
- Coverage Area Analysis
- Shortest Path Distance and Time
- Kernel Density Estimation (KDE)
- Average Nearest Neighbour (ANN)
- Ripley’s K Function

### Network Analysis
- Degree Centrality
- Closeness Centrality
- Betweenness Centrality

All analyses were conducted using **Esri ArcGIS Pro**.
